# Glamhub-Beauty-Salon-Webd 

GlamHub is a modern, pastel‑themed beauty and styling website designed to offer users a smooth, elegant, and visually appealing experience. 
The project showcases essential pages like Home, Services, Gallery, About, Login, and Sign Up, each crafted with clean HTML structure, minimalistic CSS, and intuitive navigation.
The platform highlights beauty services, displays curated galleries, and provides user authentication through well‑validated forms. With responsive layouts, soft color palettes, and polished UI elements, GlamHub reflects a professional blend of aesthetics and functionality — making it ideal for showcasing frontend design skills, user‑centric layouts, and JavaScript‑based form validation.
